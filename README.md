# Panorama Project - Yang Benjamin
## But du projet
Le but du projet est de se familiariser avec la manimupaltion géométrique dans l'espace des images. Après avoir étudié le bruit dans le cadre de mon projet sur le bruit et débruitage, il me semble intéressant d'étudier comment superposer des images, comment réaliser des transformations dans l'espace, etc...

Nous étudierons de la même manière : d'abord les expériences sur mon notebook principal, puis extraire les fonctions dans du code python au propre et créer une app streamlit en utilisant mes fonctions. C'est parti !
